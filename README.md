# Academic Research Project: Investment Perception & Selection Behavior in Mutual Funds

A comprehensive web platform designed for academic research on mutual fund investment patterns, analyzing how investors perceive and select mutual funds based on various factors. This project explores behavioral finance principles and investment decision-making processes.

## How to Run

1. Clone or download the project files
2. Open `index.html` in your web browser, or:
3. Use a local server (recommended):
   ```bash
   python -m http.server 8000
   ```
4. Navigate to `http://localhost:8000` in your browser

## Key Functionality

### Fund Comparison
- Filter funds by category, risk level, or search term
- Select up to 3 funds for detailed side-by-side comparison
- View detailed metrics including returns, expense ratios, and more

### Investment Calculator
- Calculate potential returns on mutual fund investments
- Input initial investment, monthly SIP amount, expected return, and investment period
- Get projections for total invested amount, estimated returns, and total value

### Role-Based Access
- Switch between different user roles using the dropdown in the header
- Each role has access to different features and content
- Content adapts based on the selected role

## Academic Research Focus

This project serves as an academic exploration of investment perception and selection behavior towards mutual funds. It demonstrates:

- Application of behavioral finance principles
- Analysis of investment decision-making patterns
- Implementation of modern web development practices
- Research methodology in financial technology

The platform provides a framework for studying how different user roles (investors, advisors, analysts) interact with investment information and make decisions.
